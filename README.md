# ❓ GK Quiz — Project 3

An interactive General Knowledge quiz app built with HTML, CSS, and JavaScript as part of the DecodeLabs Frontend Development Internship (Batch 2026).

## 📌 Project Overview
This project focuses on **Interactive Web Elements** — using JavaScript to manipulate the DOM and create pages that respond to user actions in real time.

## ✅ Key Requirements Met
- Buttons and toggles for user interaction
- Dynamic content updates via JavaScript
- Dark/Light theme toggle

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## 📚 Skills Demonstrated
- JavaScript basics
- DOM manipulation
- Event listeners
- Dynamic UI updates
